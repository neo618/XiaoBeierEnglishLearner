📝 语法要点

1. There be 句型 - 单数

表示"某处有某物"，当后面是单数名词或不可数名词时，用 There is...
There is a bike. 有一辆自行车。
There is a lake. 有一个湖。

2. There be 句型 - 复数

当后面是复数名词时，用 There are...
There are two beds. 有两张床。
There are many trees. 有许多树。

3. 一般疑问句变换

将 is/are 提到句首，句末加问号
There is a desk. → Is there a desk?
There are chairs. → Are there chairs?

4. 方位介词

in 在...里面 | on 在...上面 | near 在...附近
in the forest 在森林里
on the hill 在山上
near the lake 在湖附近
