# 小学生英语复习软件 📚

一个专为小学生设计的英语学习复习软件，支持单词学习、语法练习、测试和听力训练。

## 功能特点

### 📖 学单词
- 按周一到周五分类显示本周单词
- 每个单词包含：英文、音标、中文意思、例句
- 点击单词卡片可以听发音
- 已掌握的单词会显示绿色边框

### 🎯 背单词
- 卡片式背诵模式
- 点击卡片翻转查看中文意思
- 标记"认识"或"不认识"
- 自动记录学习进度

### 📝 学语法
- There be 句型（单数/复数）
- 一般疑问句变换
- 方位介词（in/on/near）
- 配套语法练习题

### ✏️ 做测试
- 10 道选择题
- 自动评分
- 成绩保存

### 练听力
- 家长读题，孩子选择答案
- 基于课本听力题型设计

### 📷 添加单词
- 支持手动添加新单词
- 未来版本支持图片识别自动添加

### 📊 看进度
- 总单词数、已掌握、学习中统计
- 学习进度条
- 需要复习的单词列表

## 使用方法

### 方式 1：直接打开
双击 `index.html` 文件即可在浏览器中打开

### 方式 2：使用本地服务器
```bash
# 如果有 Python
cd english-tutor
python -m http.server 8080

# 然后访问 http://localhost:8080
```

## 数据存储

学习进度保存在浏览器本地存储（LocalStorage）中，关闭浏览器后数据不会丢失。
